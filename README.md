# Stock Predictor using Fear & Greed Index

## Overview

This project attempts to predict stock market trends by incorporating market sentiment data, specifically the "Fear & Greed Index." This project was completed as part of the Northwestern FinTech Bootcamp.

## Key Features

* **Sentiment Analysis:** Integrates the Fear & Greed Index as a key feature to gauge market sentiment and its potential impact on stock prices.
* **Predictive Modeling:** Implements machine learning or time series models to forecast stock performance.
* **Data Analysis:** Analyzes historical stock data alongside sentiment data to identify correlations and build the prediction model.

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* Scikit-learn (or other ML/Deep Learning libraries like TensorFlow/Keras)
